Mathematical formulas


Linear Equations

A liner equation is any equation that can be written in the form.

ax+b=0

where a and b are real numbers and x is a variable.

Quadratic Equation

The standard form of a quadratic equation looks like this:

ax2+bx+c=0

where a, b, c are known values and x is the variable or unknown. Also, a can't be 0.

The area of a circle (A)

π (Pi) times the radius (r) squared:

A = π * r2

or, when you know the diameter (D):

A = ( π / 4 ) * D2

or, when you know the circumference (C):

A = C2 /4 * π

Intersection

The intersection of two sets A and B, denoted by A ∩ B, is the set of all objects that are members of both te sets A and B. In symbols,:

A ∩ B = {x : x ∈ A and x ∈ B }

That is, x is an element of the intersection A ∩ B if and only if x is both an element of A and an element of B.
